# 🎯 Quick Start - MockGen

## ⚡ **Get Started in 2 Steps**

### **Step 1: Add Your API Key**

1. Get free API key: https://makersuite.google.com/app/apikey
2. Open: `apps/server/.env`
3. Replace `your_gemini_api_key_here` with your key

### **Step 2: Run**

```bash
npm run dev
```

Visit: **http://localhost:8080**
